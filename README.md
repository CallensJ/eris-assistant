# E.R.I.S. 🤖
**Extensible Real-time Intelligent Support**

E.R.I.S. est un assistant desktop intelligent optimisé pour l'environnement **Debian**.
Il sert d'orchestrateur entre l'utilisateur et le système pour automatiser les workflows de développement web

## 🚀 Objectifs du MVP1
- Interface de chat texte (PyQt6 + QML)
- Intégration de l'API Gemini pour la compréhension des intentions
- Exécution d'actions locales (scripts Python, ouverture d'applications)
- Historique minimal des interactions (SQLite)

## 🛠️ Installation
1. Cloner le repo.
2. Installer les dépendances :   pip install -r requirements.txt
